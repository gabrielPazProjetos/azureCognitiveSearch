 Desafio de Projeto DIO

--- Descrição
Este projeto faz parte do desafio da DIO e tem como objetivo aplicar técnicas de organização e pesquisa de documentos por meio da ingestão de dados e indexação utilizando ferramentas de inteligência artificial do **Azure Cognitive Search**.  

Durante as aulas, foram abordados três passos principais:
- Ingestão de conteúdo para IA  
- Criação de índices inteligentes  
- Exploração prática dos dados organizados  
O foco está em desenvolver uma compreensão sólida sobre como essas ferramentas podem ser utilizadas para minerar e extrair conhecimento de grandes volumes de informação.  

--- Tecnologias Utilizadas
- Azure Cognitive Search  
- Azure Portal  
- Python (requests) para interação com a API REST  

--- Insights Obtidos
- Entendimento prático de como funciona a ingestão de dados no Azure Cognitive Search  
- Criação de índices inteligentes para organizar documentos  
- Aplicação de enriquecimento de IA para extrair informações relevantes  
- Realização de buscas cognitivas com filtros e ordenações  
- Compreensão de como minerar conhecimento em grandes volumes de dados
